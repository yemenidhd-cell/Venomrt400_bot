# 🇾🇪 السلفي برو للأدوات

بوت تيليجرام احترافي متكامل يحتوي على أدوات متنوعة.

## الميزات
- أدوات بايثون (تشفير/فك تشفير/ضغط/فك ضغط)
- معلومات الجذرية (Telegram/TikTok/Instagram)
- أدوات الشبكات
- أدوات الأرقام
- مولد أسماء المستخدمين
- نظام نقاط وإحالات
- لوحة تحكم للمطور

## المتطلبات
- Python 3.13+
- توكن بوت من @BotFather

## النشر على Render
1. رفع المشروع على GitHub
2. ربطه بـ Render
3. تشغيل تلقائي
